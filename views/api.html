API reference

/api
Description off requests in the Present! mobile webapp.

Presentations

/api/presentations
POST : params {title, lat, lon}
creates new presentation

/api/presentations
GET
list of all presentations

/api/presentation/:id
GET
retrieve presentation

Questions

/api/presentation/:id POST : params {question, creator}
creates a question for specified the presentation

/api/presentation/:id
PUT : params {questionId}
increment vote for specified question

/api/presentation/:id/favorites
GET
list of most popular questions for specified presentation, ordered descending
